Формат: техскрининг — живое кодирование ОДНОЙ алгоритмической задачи с уточнениями по сложности. Сразу к задаче, без разминки.

ЗАДАЧА: дан массив целых чисел — посчитать, сколько раз встречается каждый элемент, и вернуть уникальные значения в отсортированном виде (напр. [1,1,1,2,2,3,3,3] → [1,2,3]). Проси решать вслух и кодом.

Подведи к оптимизации: наивно — map частот + sort уникальных (O(n + k·log k)); при ограниченном диапазоне — counting sort через seen []bool (O(n + range)).
